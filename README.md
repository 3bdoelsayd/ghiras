# تطبيق غِراس - Ghiras App 📖🌱

تطبيق إسلامي شامل ومتميز يهدف إلى تقديم تجربة إيمانية متكاملة وسلسة للمستخدم المسلم، مع التركيز على دقة عرض المصحف الشريف وجمالية الواجهة.

## ✨ أبرز المميزات:

- **المصحف الشريف:** عرض سطر بسطر مطابق لمصحف المدينة المنورة (15 سطراً لكل صفحة) مع الرسم العثماني والخط الرسمي لمجمع الملك فهد.
- **الأذان ومواقيت الصلاة:** دقة عالية في حساب المواقيت حسب الموقع الجغرافي مع تنبيهات بصوت الأذان الكامل.
- **الختمة القرآنية:** نظام ذكي لمتابعة ختمة القرآن الكريم وتحديد الورد اليومي.
- **أذكار المسلم:** مكتبة شاملة للأذكار اليومية (الصباح، المساء، النوم) بتصميم مريح للعين.
- **تصفح القراء:** إمكانية الاستماع وتحميل تلاوات لأكثر من 200 قارئ من مشاهير القراء حول العالم.
- **التشغيل بدون إنترنت:** يدعم التطبيق تحميل صفحات المصحف وسور القراء للعمل بكفاءة في وضع الأوفلاين.

## 🚀 التقنيات المستخدمة:

- **Flutter & Dart:** لبناء واجهة مستخدم سريعة وعابرة للمنصات.
- **GetX:** لإدارة الحالة (State Management) والتنقل.
- **Hive:** لتخزين البيانات المحلية والـ Cache بسرعة عالية.
- **Just Audio:** لتوفير تجربة صوتية مستقرة واحترافية.
- **Quran.com API:** المصدر الأساسي لبيانات المصحف والكلمات.

## 🛠️ كيفية التشغيل:

1. تأكد من تثبيت **Flutter SDK**.
2. قم بعمل Clone للمستودع: `git clone https://github.com/username/ghiras.git`.
3. انتقل لمجلد المشروع: `cd ghiras`.
4. قم بتحميل المكتبات: `flutter pub get`.
5. قم بتشغيل التطبيق: `flutter run`.
